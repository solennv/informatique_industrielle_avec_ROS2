Description du projet

Pour ce travail pratique, vous allez installer une plateforme robotique ROS2 sur un Raspberry Pi (Pi5) et documenter le projet. Le projet consiste donc à:

  1. installer Ubuntu sur le Raspberry Pi,

  2. installer ROS2,

  3. tester et documenter les tests de l’installation système

  4. Créer la rerprésentation mécanique du pantographe dans un fichier URDF

  5. Créer un package ROS2 pour contrôler le pantographe

  6. Créer des tests et documenter les tests

  7. Tester en simulation avec Gazebo et RViz

  8. Tester en réel avec le pantographe

La documentation correspondra à un fork de ce projet et sera publiée sur votre github. Dans ce projet un canevas de documentation est fourni. Vous devrez le compléter et le publier. Vous soumettrez votre travail sous forme de push request à ce projet.
